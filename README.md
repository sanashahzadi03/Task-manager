# 📝 Interactive Task Manager

A sleek, lightweight, and high-performance productivity application built from the ground up using **Vanilla HTML5, CSS3, and Modern JavaScript (ES6+)**. This project delivers a lightning-fast user experience completely free of heavy third-party frameworks or external libraries.



## 🚀 Features

- **Full CRUD Functionality:** Seamlessly add tasks, delete entries, and edit existing tasks inline with automatic input focus.
- **Explicit Completion Tracking:** Intuitive custom checkbox workflow to mark tasks as completed with satisfying visual feedback.
- **Live State Filtering:** Instantly sort and toggle views between **All**, **Active**, and **Completed** tasks without page reloads.
- **Persistent LocalStorage:** Saves your task list natively in the browser so your data survives page refreshes and closed tabs.
- **Omni-Responsive Design:** Tailored using fluid layouts (`clamp()`) and mobile breakpoints to look pixel-perfect on desktops, tablets, and smartphones alike.

## 🛠️ Tech Stack

- **Structure:** HTML5 (Semantic elements)
- **Styling:** CSS3 (Flexbox, Custom Variables, Fluid Typography)
- **Logic:** Vanilla JavaScript (ES6+, DOM Manipulation, Event Delegation)
- **Storage:** Web Storage API (`localStorage`)

## 📦 Installation & Setup

Since this project uses pure, native web technologies, there is no need for `npm install` or complex build tools.
